    My name is Kyle Dodd. I am a full stack developer. I grew up in Okinawa Japan, but I now live in Iowa City, Iowa. My hobbies are skateboarding, mixed martial arts, and programming.

  • How I got into coding

    I was very intrigued about how computers and the internet work, so I did Coding Dojo. Coding Dojo is a short, intensive programming training that teaches its students to be competent in full stack web development. Coding Dojo teaches three technology stacks. Python, Java, and MEAN. I quickly grew passionate about programming during my time at this school when I began to realize how I could solve problems through creative thinking. Learning to code gave me the tools to build almost anything I could imagine.

TimeSheets

Clockin/ Clockout Application

    Functionality:

  • Login/Registration with built-in validations and password hashing
  • Clock in/out and provide feedback on day-to-day experiences
  • Allows for clocking out at an earlier time if user forgot to clock out when they were supposed to
  • Settings page lets users change their email, name, and password
  • Administrator level allows managers to view daily reports as well as delete and adjust user level of all other users

    Technologies/Tools Used:

  • Python
  • Framework - Django
  • Datebase - SQLite
  • Front End - HTML/CSS/JavaScript/BootStrap
  • Deployed on AWS EC2
